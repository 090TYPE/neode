Place font files here (the app auto-loads any *.ttf in this folder):

  JetBrainsMono-Regular.ttf   — https://www.jetbrains.com/lp/mono/
  JetBrainsMono-Bold.ttf
  JetBrainsMono-Italic.ttf
  Inter-Regular.ttf           — https://rsms.me/inter/

If no fonts are present, NeoIDE falls back to the system monospace font.
